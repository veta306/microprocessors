# To run this project
1. Create the PostgreSQL database:
- fill it by code from db.sql
- specify pg user data in `server/.env` file
2. Run server:
- Go to `server` folder
- run `npm install`
- run `npm start`
3. To run React app
- go to `client` folder
- run `npm install`
- run `npm run dev`
- go to [localhost:5173](http://localhost:5173)
